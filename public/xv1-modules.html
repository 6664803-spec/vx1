输入 / 输出契约

输入：历史范围、使用期数、是否只用 timeseq 自身数据。输出：每个位置统一 Top3 + score + confidence + reason。

回测口径

只用过去预测未来，walk-forward 滚动回测，分开统计 Top1 / Top3 / 每位置命中率 / 连续命中失误。

模块分层

把规则、统计、序列和融合拆开，每层都保留自己的 Top3 与得分。

规则评分定义

频次 + 最近热度 + 间隔 + 转移，带时间衰减，作为每位置模型的起步基线。

训练来源

xv1 的训练入口只接 timeseq_predictions，当前保留 1620 条新格式数据用于训练对接。
